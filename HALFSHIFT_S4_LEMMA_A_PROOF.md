# Coset-Uniformity, S4, and the reduction of Lemma A to one elementary lemma (2026-06-01)

This document discharges the "crux" obligation of the spectral-gap certificate. After the Session 39
research pivot, the entire upper-cascade Lemma A, the strict-upper structure (R1) and rank-1 defect
(R2) of Lemma B's foundation, and the Half-Shift Invariance lemma **all reduce by elementary algebra
to two uniform-fibre lemmas sharing one engine**: Coset-Uniformity (CU, for Half-Shift / rank-1) and
the Shell-Bijection (SB, for the upper cascade). Both are elementary properties of a
multiplication-by-unit map on a cyclic 2-group; neither is a Gauss sum. S4, which prior notes called
the analytic core, is a one-line finite geometric series.

Notation: `w = exp(2*pi*i / 2^k)`. `v2` is the 2-adic valuation. `Syr(n) = oddpart(3n+1)`. `3` is a
unit mod `2^m` for every `m` (gcd(3,2)=1); `3^{-1}` denotes its inverse in the relevant `Z/2^m`.

## 1. Two uniform-fibre lemmas (CU and SB) -- the shared engine

> **Lemma CU.** Fix `K >= 1` and an odd `x` with `v := v2(3x+1) < K`. Put `q0 := (3x+1)/2^v` (odd).
> As `m` ranges over `Z/2^K`, the residue `Syr(x + m*2^K) mod 2^K` is **uniform over the coset**
> `C := q0 + 2^{K-v} * (Z/2^K)`, a set of `2^v` residues, each attained exactly `2^{K-v}` times.

**Proof.** Two elementary steps.

*(a) Valuation is frozen and Syr is affine in `m`.* For `n = x + m*2^K`,
```
3n+1 = (3x+1) + 3m*2^K = 2^v*q0 + 3m*2^K = 2^v * (q0 + 3m*2^{K-v}).
```
Since `v < K`, `K-v >= 1`, so `3m*2^{K-v}` is even and `q0 + 3m*2^{K-v}` is odd (q0 odd). Hence
`v2(3n+1) = v` exactly, and as integers
```
Syr(n) = (3n+1)/2^v = q0 + 3m*2^{K-v}.
```

*(b) The multiplication map has uniform fibres onto a 2-subgroup.* Reduce mod `2^K`. The map
`phi: Z/2^K -> Z/2^K`, `phi(m) = (3*2^{K-v}) * m`, is a group homomorphism with `c := 3*2^{K-v}`,
`v2(c) = K-v` (3 is a unit). Its image is the subgroup generated by `gcd(c, 2^K) = 2^{K-v}`, namely
`2^{K-v}*(Z/2^K)`, of order `2^K / 2^{K-v} = 2^v`; its kernel `{m : 2^v | m}` has order `2^{K-v}`, so
every image element has exactly `2^{K-v}` preimages. Therefore `3m*2^{K-v} mod 2^K` is uniform over
`2^{K-v}*(Z/2^K)` with multiplicity `2^{K-v}`. Adding the constant `q0` shifts this to the coset `C`.
QED

CU uses only: `3` is a unit mod `2^K`, and the image/kernel of multiplication by `c` on a finite
cyclic group. No Gauss sums, no induction, no k-dependent error term. Verified: 0 violations,
`audit_halfshift_s4.py` test (1), all odd `x`, both parities, `v` up to `K-2`.

### Shell-Bijection (SB) -- the instance the upper cascade actually uses

CU varies the additive lift `m` of a fixed `x`; the cascade (Section 4) needs a different instance of
the same engine, varying the base point `r` across a valuation shell. State it on its own.

> **Lemma SB.** Fix `1 <= j <= k-2`. The map `psi(r) := (3r+1)/2^j mod 2^{k-j}` is a **bijection**
> from the shell `S_j := { r : 0 < r < 2^k, r odd, v2(3r+1) = j }` onto the odd residues mod `2^{k-j}`.

**Proof.** Well-defined: `v2(3r+1) = j` makes `(3r+1)/2^j` odd. Injective: `psi(r) = psi(r')` gives
`2^j q ≡ 2^j q' (mod 2^k)`, i.e. `3r+1 ≡ 3r'+1 (mod 2^k)`, so `r ≡ r' (mod 2^k)` (3 a unit), hence
`r = r'` (both in `[HalfShiftInvariance_DRAFT.md](HalfShiftInvariance_DRAFT.md),
and CU is its step 3 (`coset_uniform`), the only step the draft flagged as nontrivial. The rank-1
S_odd theorem follows as in the draft: every non-`r*` column vanishes by half-shift invariance, and
`r*` is the unique residue with `v2(3r*+1) >= k` (a unique `-3^{-1}` solution), so exactly one column
is nonzero.

## 3. S4 dead-band -- a finite geometric series

Define `Sodd(alpha, m) := sum_{q odd, 0 <= q < 2^m} w^{alpha*q}`. With `q = 2l+1`:
```
Sodd(alpha, m) = w^alpha * sum_{l=0}^{2^{m-1}-1} (w^{2*alpha})^l = w^alpha * G,   G := geometric sum, ratio zeta = w^{2 alpha}.
```

> **Lemma S4.** For `m >= 1`:
> - if `2*alpha ≡ 0 (mod 2^k)` (i.e. `alpha in {0, 2^{k-1}}`): `Sodd = 2^{m-1} * w^alpha`, modulus `2^{m-1}` (FULL);
> - else `Sodd = w^alpha (w^{2^m alpha} - 1)/(w^{2 alpha} - 1)`, which **vanishes iff** `k - m <= v2(alpha) <= k - 2`;
> - else (`v2(alpha) < k - m`) it is a nonzero partial sum.

**Proof.** If `zeta = w^{2 alpha} = 1` (i.e. `2^{k-1} | alpha`), the `2^{m-1}` terms are all `1`, sum
`= 2^{m-1}`. Otherwise the geometric sum is `(zeta^L - 1)/(zeta - 1)`, `L = 2^{m-1}`, with
`zeta^L = w^{2 alpha * 2^{m-1}} = w^{alpha * 2^m} = exp(2 pi i * alpha / 2^{k-m})`. This sum is `0` iff
`zeta^L = 1` and `zeta != 1`, i.e. `2^{k-m} | alpha` (so `v2(alpha) >= k-m`) and `2^{k-1} nmid alpha`
(so `v2(alpha) <= k-2`). QED

Pure undergraduate algebra. The only boundary to guard in Lean is `m >= 1` (here `m = k-j >= 2` since
`j <= b <= k-2`). Verified: 0 fails, all `alpha`, `m = 2..k-1`, both parities (test (2)).

## 4. Lemma A (clean) reduces to CU + S4 + elementary counting

Block `B = P_a U_clean P_b`, target level `a`, source level `b`, `a < b`, `d := b - a`. The
character-basis entry (target `xi`, `v2(xi)=a`; source `eta`, `v2(eta)=b`) is, up to the `1/N`
normalisation (`N = 2^{k-1}`),
```
hat(eta, xi) = sum_{r odd, v(r) <= b, v(r) < k} w^{eta*q(r) - xi*r}.
```

**(S1=SB) group by `j := v(r)`, `j = 1..b` (the mask `[v(r) <= b]` caps `j` at `b`, and `b <= k-2`).**
For fixed `j`, `3r+1 = 2^j*q` with `q = Syr(r)` odd, so `r = (2^j q - 1)/3` (well-defined, 3 a unit)
and
```
eta*q(r) - xi*r = eta*q - xi*(2^j q - 1)/3 = q*alpha_j + xi*3^{-1}  (mod 2^k),   alpha_j := eta - xi*2^j*3^{-1}.
```

**Valuations, computed once (the prerequisite for the reduction AND the survivor rule).** With
`v2(eta) = b` and `v2(xi*2^j*3^{-1}) = a+j`:
```
v2(alpha_j) = a+j   (j < d);     v2(alpha_j) = b   (j > d);     v2(alpha_j) >= b   (j = d).
```
In every case `v2(alpha_j) >= j` (using `a >= 0` for `j < d`, and `j <= b` for `j >= d`). Therefore
`w^{q*alpha_j}` is invariant under `q -> q + 2^{k-j}`, i.e. depends only on `q mod 2^{k-j}`. By **SB**,
as `r` runs over the shell `{v(r)=j}`, `q mod 2^{k-j}` runs over all odds mod `2^{k-j}` **exactly once**
(bijection, multiplicity 1 -- no constant to carry). Hence the `j`-shell contributes exactly
```
A_j = w^{xi*3^{-1}} * Sodd(alpha_j, k-j).
```
(`v2(alpha_j) >= j` is exactly what makes the shell sum land in `Sodd`'s canonical range `[attack1_lemmaA_proof.py](attack1_lemmaA_proof.py).)

**(S5) only `j = d` survives.** Place the valuations above against S4's dead-band `[j, k-2]`:
- `j < d`: `v2(alpha_j) = a+j`, and `j <= a+j <= b-1 <= k-3`, in `[j, k-2]` => `A_j = 0`.
- `j > d`: `v2(alpha_j) = b`, and `j <= b <= k-2`, in `[j, k-2]` => `A_j = 0`.
- `j = d`: `v2(alpha_d) >= b`; in the band (=> `A_d = 0`) unless `v2(alpha_d) >= k-1`, i.e.
  `alpha_d in {0, 2^{k-1}}`, where S4 gives the FULL value `|Sodd(alpha_d, k-d)| = 2^{k-d-1}`.

So `hat(eta, xi) != 0` iff `alpha_d = eta - xi*2^d*3^{-1} in {0, 2^{k-1}}`, and then
`|hat(eta, xi)| = 2^{k-d-1}`.

**(S6/S7 diagonal) each `eta` has exactly `2^d` owners.** `alpha_d in {0, 2^{k-1}}` means
`xi*2^d ≡ 3*eta` or `≡ 3*eta + 3*2^{k-1} (mod 2^k)`. With `v2(3 eta) = b = a+d`, each branch is a
single residue class mod `2^{k-d}` for `xi`, contributing `2^{d-1}` solutions `xi` of valuation `a`
in `[0, 2^{k-1})`; the two branches are distinct (differ by the half-modulus), total `2^d`. Diagonal:
```
G[eta,eta] = (1/N^2) * (#owners) * |hat|^2 = (1/2^{2(k-1)}) * 2^d * (2^{k-d-1})^2 = 2^{-d}.
```

**(S8 off-diagonal) disjoint ownership.** For fixed `xi`, the two candidate sources are
`eta ≡ xi*2^d*3^{-1} (mod 2^{k-1})` and that `+ 2^{k-1}`; they differ by the half-modulus `2^{k-1}`,
and the level-`b` range `[0, 2^{k-1})` is exactly half the modulus, so **exactly one** lies in range.
Hence each `xi` is owned by a unique `eta`, the columns of the support have disjoint rows, and
`G[eta, eta'] = 0` for `eta != eta'`.

**Conclusion.** `B*B = G = 2^{-d} I`, so `||P_a U_clean P_b||_2 = 2^{-d/2}` EXACT, for all `k` and all
`0 <= a < b <= k-2`. The upper-cascade row-sum is then the exact geometric series
`sum_{d>=1} 2^{-d/2} 2^{-d} = 2^{-3/2}/(1 - 2^{-3/2}) = 0.5469`, constant in `k`. QED (Lemma A, clean)
